# FreeDay Landing Page

A React landing page for FreeDay, an AI-powered system that monitors Tamil Nadu and Puducherry news channels live on YouTube, uses OCR to detect school holiday announcements, verifies them with Gemini AI, and sends WhatsApp/call alerts to parents and students.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production site is generated in `dist/`.

## Publish With GitHub Pages

1. Push this root folder to a GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Set **Source** to **GitHub Actions**.
4. Push to the `main` branch. The included workflow will build and deploy the site.
